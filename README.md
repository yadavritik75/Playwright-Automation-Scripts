# 🚀 Playwright Automation Framework

This repository contains end-to-end UI automation scripts built using **Playwright**.  
The project demonstrates strong hands-on experience in modern test automation practices.


## 📌 Tech Stack
- Playwright (JavaScript)
- Node.js
- VS Code
- Git & GitHub



## ✅ Features Covered
- 🔹 Dynamic & Static Dropdown Handling
- 🔹 End to End Automation Event Booking Tickets
- 🔹 Auto-suggest dropdowns  
- 🔹 Alerts & Popups handling  
- 🔹 Frames handling  
- 🔹 Calendar selection  
- 🔹 End-to-End user flows  
- 🔹 Locator strategies (CSS, XPath, getByRole, getByText)  
- 🔹 Screenshot & Visual Testing  
- 🔹 Page interactions & validations
- 🔹 End to End Automation Ecommerce Using getLocators

---

## ▶️ How to Run Tests

```bash
# Install dependencies
npm install

# Run all tests
npx playwright test

# Run specific test
npx playwright test tests/dropDown.spec.js

# Run in headed mode
npx playwright test --headed

#Run in debug mode
npx playwright test tests/className --debug

#Run with Test Runner
npx playwright test tests/className --ui

#Record & PlayBack Tool
npx playwright codegen amazon.com

📸 Reporting
npx playwright show-report
